# AutoRec

* Цель: споектировать и реализовать распознователь лексем на основе заданных регулярных выражений

* Алфавит: буква, цифра, подчёркивание, другие символы.<br/>
* Регулярне выражения:<br/>
  1. identifier ::= [a-zA-Z_][a-zA-Z0-9_]*<br/>
  2. number ::= [0-9]+<br/>

* Состояния:<br/>
  (s) — начальное (вне токена)<br/>
  s(id) — первая буква или подчёркивание<br/>
  m(id) — допускающее состояние для идентификатора<br/>
  s(num) — первая цифра<br/>
  m(num) — допускающее состояние для числа<br/>
  (error) — ошибка<br/>
  (e) — конец строки<br/>

* Переходы:<br/>
  (s) -> s(id)<br/>
  (s) -> s(num)<br/>
  (s) -> (error)<br/>
  (s) -> (e)<br/>
  
  s(id) -> (s)<br/>
  s(id) -> m(id)<br/>
  s(id) -> (e)<br/>
  s(id) -> (error)<br/>
  
  m(id) -> (s)<br/>
  m(id) -> m(id)<br/>
  m(id) -> (e)<br/>
  m(id) -> (error)<br/>
  
  s(num) -> (s)<br/>
  s(num) -> m(num)<br/>
  s(num) -> (e)<br/>
  s(num) -> (error)<br/>
  
  m(num) -> (s)<br/>
  m(num) -> m(num)<br/>
  m(num) -> (e)<br/>
  m(num) -> (error)<br/>
